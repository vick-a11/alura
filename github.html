    Comece criando um novo arquivo ou carregando um arquivo existente . Recomendamos que cada repositório inclua um README , LICENSE e .gitignore .

    …ou crie um novo repositório na linha de comando

    echo "# vick-a11" >> README.md 
git init 
git add README.md 
git commit -m "primeiro commit" 
git branch -M main 
git remote add origin https://github.com/vick-a11/vick-a11.git
 git push -u origin main

    …ou envie um repositório existente a partir da linha de comando

    git remoto adicionar origem https://github.com/vick-a11/vick-a11.git
 git branch -M principal 
git push -u origem principal
